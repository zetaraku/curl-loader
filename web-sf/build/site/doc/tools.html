RECOMMENDED TOOLS

This page reflects our personal experience and preferences.

A very extensive collection of open-source testing tools could be found at the site opensourcetesting

Apache JMeter is a good open-source HTTP/S loading and performance testing tools competing with curl-loader.

SIPp is an excellent open-source SIP protocol loader.

Some good commercial application loading and testing tools (but they are very expensive) are here:
  1. RadView WebLoad
  2. Mercury Interactive (HP) Tools
  3. Spirent Avalanche
  4. IXIA IxLoad
