CURL-LOADER THANKS

Thanks to the creators and maintainers of the packages, we are using:
- libcurl (Daniel Stenberg) for the HTTP 1.1, FTP, telnet, ldap, etc client stacks;
- openssl for the TLS/SSL stack;
- dialog for the console dialog manu;
- libevent (Niels Provos) for the I/O event demultiplexing framework;
- iprouted2 and linux netdev for netlink and linux TCP/IP stack (Alexey Kuznetsov, David Miller and many others).

The list of people, that provided assistance, patches, bug-fixes, ideas is in the THANKS file of the curl-loader tarball.
